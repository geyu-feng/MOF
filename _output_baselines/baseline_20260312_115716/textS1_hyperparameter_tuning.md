Text S1. Hyperparameter tuning

Specifically, the first step involved determining the range of each hyperparameter and selecting several parameter values within the range using reasonable intervals. Subsequently, an exhaustive search method was employed to explore all possible combinations of hyperparameters, and 10 models were trained on different training sets (Fig. S7). Afterwards, each model was applied to predict the validation set and the coefficient of determination was calculated. Finally, the average coefficient of determination was calculated for each combination of hyperparameters, and the combination yielding the highest coefficient of determination represented the optimal hyperparameter combination for the machine learning (ML) model (Fig. S8). This strategy was applied independently to optimize the hyperparameters for each ML model during training. After a series of parameter tuning iterations, each ML algorithm trained its model on the training set using its respective optimal hyperparameter combination [1, 2].
